HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • Html is the frame of our website
  • Html elements are contained in "< >"
  • Html attributes should always be declared at the beginning of an element
  • meta is a special element that describes the website to help web broswers look for and handle your webpage
  • Elements are pieces of code to preform a certain task on an html page
  • Element can contain other elements inside of them making them a parent elemnet. The elements inside the parent are referred to as child elements.
  • Semantic Html elements are elements that describe its content and role in the webpage

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: This adds all changes to a place ready but not on the local branch your working on
  • git commit -m "description": Confirms changes that you have made on your local device and we can add a comment to describe the changes made in the branch
  • git pull origin main: This will confirm that you are working on the most updated version of the main project your modifying
  • git push origin Branch_Name: We will be changing the github branch that is online to match our local devices branch

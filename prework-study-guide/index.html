HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • Html is the frame of our website
  • Html elements are contained in "< >"
  • Html attributes should always be declared at the beginning of an element
  • meta is a special element that describes the website to help web broswers look for and handle your webpage
  • Elements are pieces of code to preform a certain task on an html page
  • Element can contain other elements inside of them making them a parent elemnet. The elements inside the parent are referred to as child elements.
  • Semantic Html elements are elements that describe its content and role in the webpage

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • We can have more than one selector for one declaration block to reuse code
  • The special character * in a css file referres to all elements in the webpage
  • There are three different ways to code css which is inline css, internal css, and external css
  • Internal style sheet is styling for a single webpage and all of the code is in the head section of the html file
  • In internal css coding we can configure css class and IDs to style multiple elements
  • External styling is where you have a separte file and is preferred in most cases
  • Link is the keyword for connecting external css files with your html file

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: This adds all changes to a place ready but not on the local branch your working on
  • git commit -m "description": Confirms changes that you have made on your local device and we can add a comment to describe the changes made in the branch
  • git pull origin main: This will confirm that you are working on the most updated version of the main project your modifying
  • git push origin Branch_Name: We will be changing the github branch that is online to match our local devices branch

JavaScript

  • JavaScript is a tool to make a website more interactive
  • Arrays are not limited to one datatype
  • Datatypes in JavaScript are not strict and can be used weakly
  • Lines of code in JavaScript end with ";"
  • "if" is a special keyword to make conditional statments in your project
  • if there is more than one conditional statement we are checking for we would use "else if"
  • Conditional Statements should end with an else statement which is like a final error catch assuming none of the other conditional statements are false
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • We can make functions to orgonize our code to preform a set task
  • We must call functions or else we wont preform anything
